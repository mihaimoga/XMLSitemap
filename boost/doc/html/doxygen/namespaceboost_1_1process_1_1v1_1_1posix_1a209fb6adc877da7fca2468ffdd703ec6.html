Global use_vfork

boost::process::v1::posix::use_vfork

Synopsis

// In header: <boost/process/v1/posix.hpp>

unspecified use_vfork;

Description

This property will replace the usage of fork by vfork.

Note

vfork is no longer an official part of the posix standard.
