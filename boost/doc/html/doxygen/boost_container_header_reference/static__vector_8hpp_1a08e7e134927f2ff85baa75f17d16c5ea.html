Function template erase

boost::container::erase

Synopsis

// In header: <boost/container/static_vector.hpp>


template<typename T, std::size_t N, typename O, typename U> 
  static_vector< T, N, O >::size_type 
  erase(static_vector< T, N, O > & c, const U & v);

Description

Effects: Erases all elements that compare equal to v from the container c.

Complexity: Linear.
